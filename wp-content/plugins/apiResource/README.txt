== Description ==

This plugin fetch books and categories data with REST API.

1. Upload `api-resource` directory to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Place `<?php do_action('apiResourceHook'); ?>` in your templates
